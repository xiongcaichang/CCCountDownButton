# CCCountDownButton
倒计时按钮,用于获取验证码倒计时


* 使用方法  导入头文件(Import the header file)

```
#import "UIButton+countDown.h"
```

* 开始倒计时   (begin countDown)

```objc
 [button countDownWithTimeInterval:10];
```
